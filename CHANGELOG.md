# 変更履歴（CHANGELOG）

このアプリの版ごとの変更点をまとめます。新しい版を入れるかどうかの判断材料にしてください。
新しい版が上にくるように、上へ追記していきます。

書き方のめやす（各版で、当てはまるものだけ書けばOKです）：

- **追加** … 新しくできるようになったこと
- **改善** … これまでの動きが良くなったこと
- **修正** … 不具合を直したこと
- **注意** … 更新時に気をつけること（あれば）

各校での更新は、`src/App.jsx` を全選択→全置換で貼り替えるだけです。
ヘッダーやセルの描画など構造に関わる更新は、本番に直接入れず、まずブランチを作ってプレビューURLで確認してください。

## [未リリース]

ここに、次の版に入れる予定の項目を書きためていきます。

### 既知の不具合（次回対応予定）

**授業者一覧ビューで、コマを別の曜日へ移動できない（例：水2限の社会を火2限へ）**

- **症状：** 授業者一覧ビューでコマをつかみ、別の曜日（日タブ）へ寄せて移動しようとすると、置けそうに見える緑のセルで指を離しても何も起きず、コマが元に戻る。
- **補足：** 他の先生の列が 🚫 になるのは仕様（そのコマはその先生の列にしか置けない）。問題は「別の曜日へ落とすと成立しない」点。
- **原因の見立て：** 授業者一覧ビューは先生×時限を1日ずつ表示し、ドラッグ中に日タブへ寄せて曜日を切り替える作り。タブ切り替えをはさむと移動元と移動先の対応が取れず成立しない。更新とは無関係の前からの挙動。
- **回避策（これで移動できます）：** 学級別ビューに切り替える。曜日が横の列に並ぶので、対象の学級を選び、水曜2限のコマを火曜2限のセルへそのままドラッグすれば移動できる。
- **直す方針（メモ）：** ドラッグ中の日タブ切り替え時の src/tgt の取り回し。中核のドラッグ処理のため、再現条件を固め、必ずプレビューで検証してから本番へ。

**学級別ビューの「出張の見え消し」は、振替済み判定が未対応（v8_7_13 は授業者一覧ビューのみ対応）**

- **症状：** 学級別ビューでは、出張の授業を待機・別時限へ動かしても、見え消し（灰色）が消えずに残ることがある。
- **状態：** 授業者一覧ビューは v8_7_13 で対応済み。学級別ビューは日付モードと基本モードでキー（曜日／日付）が異なるため、同じ手当てを別途・慎重に入れる。

- **修正：** 学級別ビューで、合同授業やA/B週分割のコマなど、一部の重複が表示されない取りこぼしがあったのを修正（base と changes の両方を見て判定するようにした）。
- **追加：** ヘッダーの「⚠ 重複 ◯件 ▸」がボタンになり、押すたびに重複している箇所へ1つずつ移動し、該当のコマを数秒間点滅して示すようにした。どの画面からでも、押していくだけで全部の重複を順番に確認できる。
- **改善：** 重複件数の数え方を「重複している箇所の数」に変更（同じコマに2つ重なっていても1箇所と数える）。ボタンを押す回数と数字が一致して分かりやすくなった。

（※ 開発途中で「マウスを乗せると自動で移動」する動きも試したが、紛らわしいため最終的に外し、ボタンで移動する方式に統一しました。）
